rb-gsl by JamesHarrison

A ruby 1.9.1 compatible fork of rb-gsl bindings for the Gnu Scientific Library

The Gnu Scientific Library is a powerful library for doing various mathematical operations. These bindings let you use GSL from within your Ruby apps.

Dependencies

Gnu Scientific Library (apt-get install libgsl0-dev on Ubuntu) gsl-config on PATH

Install

(For Ubuntu: YMMV for other distros: apt-get install libgsl0-dev) git clone git://github.com/JamesHarrison/rb-gsl.git cd rb-gsl ruby setup.rb config ruby setup.rb setup ruby setup.rb install
